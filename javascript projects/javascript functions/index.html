javascript functions

  1. simple functions
    2. a function in javascript are the block of code designed to perform a particular task.
    syntax of the function looks something like this:: 
    function name(params (variable))
                 {
                    function body.
            }

    EXAMPLE:: 

    function naam(x, y, z) {
                    console.log("mera naam " + x + " hai");    
                    console.log("mera naam " + y + " hai");    
                    console.log("mera naam " + z + " hai");    
                }
                
                naam("yaseen", "umair", "uzair");
    OUTPUT:: 
    mera naam yaseen hai
    mera naam umair hai
    mera naam uzair hai

  2. function with parameters.
    3. function function name(parameter 1, parameter 2)      //here the parameters behave as local variables. that are temporary.

    EXAMPLE:: 

    function sum(a, b){
                    console.log(a + b);
                }
sum(34, 6);     // calling the function
    now in this code the 34 will take the place of "a" and 6 will take the place of "b";

    OPTIONAL PARAMETERS:

    we can also predefine the any parameter while making the function this parameter will be optional so that if we enter the three numbers it will be stored in third variable but if we do not enter any of the number but we have predefined that parameter then the code of the function will use the predefine parameter of the function.

    EXAMPLE:: 

    function addition(a = 3, b = 4 , c = 6) {
                    console.log(a, b , c);
                    console.log(a + b + c);
                }
                addition(34, 55, 30);

    OUTPUT:
    34 55 30
    119

    OR: 
    function addition(a = 3, b = 4 , c = 6) {
                    console.log(a, b , c);
                    console.log(a + b + c);
                }
                addition();
    OUTPUT:
    3 4 6
    13

  3. variable as a function.
    4. we can make the function in a variable.
    A function can also return a value that value is returned to the caller.

    SYNTAX:: 
    const function name = ( a , b (these are variable names) ){
                // function body  and declaring the new variable which is to be returned.
                // return the declared variable. 
                }
            calling the function in a variable.
    EXAMPLE:: 

    const add = (a,b) =>{
                    let c = a+b;
                    return c;
                }
                
                let y = add(36,4);
                console.log(y);
