javascript variables.

  1. variables (var)

    2. variables in javascript are just like the variables in c & c++ these variables can store the values such as numbers, charecters, floats, doubles and many more..

    variabls in javascript are declared in single line as well as in multiple lines.
    to declare the variables in the single line we use the coma ',' separeted values:
    EXAMPLE:: var a, b;
    we can declare the variables of javascript in multiple lines also.
    EXAMPLE:: var a;
                          var b;

    also we can assign the value of the varible in the declaration or outside the declaration also. EXAMPLE:: var a = 34, b = 90;
    EXAMPLE:: var a , b;
                          a = 34;
                          b = 45;
    note: the semi colon ";" after all the lines is mendarory.
