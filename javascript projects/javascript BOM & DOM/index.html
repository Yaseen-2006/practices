  3. console object method (console.(method-name))
    1. assert()
      2. this is used to assert a condition.

      The .assert() method in JavaScript is used for testing conditions in code, typically for debugging purposes. It checks whether a condition evaluates to true, and if it doesn't, it outputs an error message to the console. If the condition is true, nothing happens, and the code continues as normal.

      a = 45
b = 34
console.assert(a>b, "the equation sucks O.O")
      OUTPUT: 
      Assertion failed: the equation sucks O.O
      this meant this checks for the condition. if false then prints the output as assertion failed.
    2. .clear()
      3. this clears the console and it is used in the console.
    3. .log()
      4. prints the output on the screen.
    4. .table()
      5. console.table() is a method in JavaScript that allows you to display tabular data in a table format in the browser's console. It's useful for logging arrays, objects, or arrays of objects in a structured and easy-to-read way.
    5. .warn()
      6. this shows a warning in the browser.
    6. .error()
      7. this is used to display an error in the console.
    7. .info()
      8. this is used to display an additional info in the console.
