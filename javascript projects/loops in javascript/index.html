  4. WHILE loop.
    5. SYNTAX:: 

    (initialization)
    while (condition){
            code to be executed.
                }

    this loop is used to execute a certain task at a certain condition wich will be given in the brackets after the "while". 

    let i = 2;                            // initialization
    while (i>=10){                    // condition which is to be satisfied.
                console.log(i);       // prints "i"
                i++;                  //this line increses the value of i eg. 3,4,5,6,....
                }

    the above code demonstrates that the how while loop works.

    step 1 :: the value of i is initialized as i = 2
    step 2 :: the while loop is started whith the condition till where the code will be executed. i.e whenever the value of i will be 10 the code will run.
    step 3 :: the output "i" is printed.

    now according to the given condition the value of the i will be printed when till it's value becomes greater than or equal to 10.

    NOTE:: The program will be infinite and will never end if the condition of the while loop never meets.
