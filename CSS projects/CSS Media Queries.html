CSS media Queries.

    media queries in css are used to make the website responsive.

  1. @media not | only mediatype and (expressions)
    css-code;
    for example::
    @media means that we are writing a media query. not | only means that this condition will not or only apply on the given mediatype. mediatype that is screen or any other mediatype. and then in the bracket css-code is included for that meadiatype to make that website responsive. (expression) this includes the given expression such as max-width, max-height, etc.

    landscape orientation is when the width is greater than the height.
