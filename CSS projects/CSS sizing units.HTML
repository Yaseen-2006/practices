CSS sizing units

  1. px (pixels)

    aquires the Given pixels of the monitor

  2. viewport width (vw)

    it takes the Given percentage of the viewport width
    i.e example: width:100vw says:-this element will take the 100% viewport width of the device in which the website is getting previewed

  3. margin: auto;

    the margin from the left and right will be automatically adjusted so that the div will be alingedd in the center if any value in pixels is written before the auto in the margin the px value will be of height. for more refer the video @ 6:20.

  4. viewport height (vh)

    it takes the Given percentage of the viewport height i.e example: height:100vh says:- this element will take the 100% viewport height of the device in which the website is getting previewed

  5. Equal to the current font size (em)

    the em is the multiple of the font size of trhe current font size
    for example: 1em is font size is 1 times the current font size
    and 2em is the 2 times the current font size     (16 pixels is the default font size)
    applied font size is 1.5 em that will be 16*1.5=24 px (shown into the computed value)

  6. root em (rem)

    Relative to the font size of the root element.
    1em is font size is 1 times the current font size and 2em is the 2 times the current font size (16 pixels is the default font size for the rem )
    applied font size is 1.5 rem that will be 16*1.5=24 px (shown into the computed value)
    note: in this HTML page the font size is changed to x-large i.e 24px.

  7. v min

    in the computer vmin is equivalent to vh and in the phone vmin is equivalent to vw .
    this is because it computes vmin = minimum viewport therefore in computer the minimum viewport is height and in phone the minumum viewport is the width.
    e.g: if it is written that width = 100vmin then in computer it will compute as 100 vh and in the phone it will compute as vw

  8. v max

    in computer v max is equivalent to vw because maximum viewport is width and in the phone the v max is equivalent to vh because in phone maximum viewport is the height
    e.g: if it is written that width = 100vmax then in the computer the v max is equivalent to vw beacause the maximum viewport is the width and in the phone vmax is equivalent to vh because the maximum viewport is the height.

  9. min-height & width

    specifies that the given height should be the minimum height of the container example: this requires the space no matter the text is fulfilled or not. and maximum has no limit. it will keep increasing as the content adds.

    same for the width.
