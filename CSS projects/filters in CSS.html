CSS Filters

CSS Filters are applied on the images to change the contrast, brightness, saturation and many more which things we can do from the camera of the phone.

  1. filter: blur(values in px)
    2. using this property we can adjust the blurr of the image that is given.
    The values entered are in px

    EXAMPLE::


  2. filter: brightness(value in % or only numbers.);
    3. using this property we can increase or decrease the brightness of the image.
    this value can be inserted in the % or in only pure numbers.
    to if we enter the brightness level in the % then the brightness will increase after the 100%. below the 100% brightness the brightness will start decraesing as we decrease the numbers in %.

    EXAMPLE:: values are commented, filter: brightness(10); and filter: brightness(50%); we can check.

  3. filter: contrast(value in % or only numbers. );
    4. using this property we can increase or decrease the contrast of the image.
    the value of this is entered % or in pure numbers.
    if the value entered in % is below than 100% then contrast will start decreasing, and boyond 100% the contrast will increase.

    EXAMPLE:: values are commented, filter: filter: contrast(10); and filter: contrast(40%); we can check.

  4. filter: drop-shadow(horizontal-distance vertical-diatance blurr-value shadow-color.);
    5. using this property we can make the shadow of the image and adjust it's appearence.
    this property contains the horizontal-distance, vertical-diatance, blurr-value, shadow-color these all the properties are entered in px.
  5. filter: grayscale();
    6. using this property we can transform the image into the grayscale and it's grayscale can be adjusted If any value is not entered the picture will become fully black and white
    to adjust it's grayscale the values in % are entered.

    EXAMPLE:: filter: grayscale(50%);

  6. filter: hue-rotate(Value in deg (can be adjusted from browser.));
    7. using this property we can rotate the hue i.e (the color combinations we see). this property's value is entered in deg. THE ORIGINAL PICTURE IS DISPLAYED ABOVE.

    EXAMPLE:: filter: hue-rotate(180deg);

  7. filter: opacity(value in % );
    8. this property is used to reduce the opacity of the image which is mostly used to overlap the items which is mostly used with the help of the CSS grid. And tjis property is applied in %.
    uf this property is applied then the item behind it will be visible.

    EXAMPLE:: filter: opacity(50%);

  8. filter: saturate(Value in % or in pure numbers. );
    9. this properyty is used to adjust the saturation of the image. the value is entered in % or in pure numbers.
    if the value entered in the % is below 100% then the saturation will reduce.

    EXAMPLE:: filter: saturate(5);

  9. filter: sepia(Value in % or pure numbers. );
    10. this property is used quite similar to grayscale this property makes the pictures looks like something brownish shade. the value are entered in % or in pure numbers.

    EXAMPLE:: filter: sepia(9);


  10. filter: invert(value in numbers );
    11. This property makes the colors of the picture swapping the values of each pixel's color channels, so that light tones appear dark and dark tones appear light.

    EXAMPLE:: filter:invert(1) ;
