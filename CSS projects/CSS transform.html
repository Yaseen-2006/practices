CSS transform property.

  1. transform: rotate( given deg (degree));
    2. in this property we can rotate the given item by the given degree clockwise. i.e
    example::
    • transform: rotate( 40deg);

    this means that the item will be rotated by 40 degrees.
    example.
    box

    • transform: rotate( 0.25turn);

    this means that the item will turn 180 gerees because. 
                1 turn = 360 deg
therefore,  1/2 = 0.5 turn = 180 deg
            1/4 = 0.25 turn = 90 deg.
    this is why the item will turn 90 degrees.
    example::
    box

    • transform: rotatex( 40deg) rotatey( 40deg) rotatez( 40deg);

    using this syntax we can rotate the item with respect to the x, y and z axes.

    example:: rotated 40 degree on x axis.
    box

    example:: rotated 40 degree on y axis.
    box

    example:: rotated 40 degree on z axis.
    box
  2. transform: scale, scalex, scaley, scalez((times))
    3. in this scale property the will increase in size that much times which number will be entered in that bracket.
    example:: transform: scale(1.6);
    this means that the item will get bigger 1.6 tims it's current size.
    example:: transform: scale(1.6);
    box

    example:: transform: scalex(1.6);
    box

    example:: transform: scaley(1.6);
    box

    example:: transform: scalez(1.6);
    box
  3. transform: skew(given degree);
    4. in this property item appears like this.::
    example::
    box

    the skew only gets applied on the x and y axes it is not applied on the z axis.
  4. transform: ranslate(given percentage);
    5. using this property we can change the position of the elements in the x, y and z frames. generally this property is used to give the gap from top and bottom.
    example::
    box
  5. transform-origin: value in percentage or absolute values;
    6. using this property we can transform the origin of the box. The box rotates about its point of rotation i.e origin and we can change the point of rotation using this property.
    example:: transform-origin: 0% 0%;
    box
