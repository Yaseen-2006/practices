Object-fit and Objet-position

these all the properties are related to the images and background images.
how can we fit that picture in the particular box without ruining it's aspect ratio.

  1. object-fit
      2. the default value of the object-fit property is fill.
    • object-fit: cover;
      • this property will fit the image in the given width and height whicch is given to the image while mantaining the aspect ratio of the image.

      EXAMPLE::

      in the above picture the aspect ratio is ruined but in this picture the object-fit: cover property is applied that's why the image is containing it's aspect ratio but the surrounding of the picture is croped.

      SYNTAX:: 
      .img1{
            width: 422px;
            height: 422px;
            object-fit: cover;
                }
      note:: the width and height of the parent div doesn't affect this property.


    • object-fit: contain;
      • in this property the full image will be shown in the given width and height even if the given width and height is not fully acquired by the image.
      EXAMPLE::

  2. object-positon.

    3. this property is applied if the property object-fit: cover; is applied and after that because this property is directly related to object-fit property.

    • object-positon: top right, bottom right, top left, bottom left, center.
      • in this property the given values only are executed. which means only the described portion of the image will be seen if the property object-fit: cover is applied.

      this have possible values: top right, bottom right, top left, bottom left and center.

    • object-position: (value in px from top) (value in px from left);
      • this property value makes the distance between the image and it's given width and height from the top and left (for bottom and right we have to give the value in negatives.)

      EXAMPLE::

      SYNATX::

      .img7{
                    width: 322px;
                    height: 422px;
                    object-fit: cover;
                    object-position: 12px 24px;
                }

      SYNATX::

      .img7{
                    width: 322px;
                    height: 422px;
                    object-fit: contain;
                    object-position: 12px 24px;
                }
    • object-position: (value in %)(value in %);
      • also we can enter the values in % to make the same changes as above.

      SYNTAX::

      .img9{
                    width: 322px;
                    height: 422px;
                    object-fit: cover;
                    object-position: 12% 5%;
                }
