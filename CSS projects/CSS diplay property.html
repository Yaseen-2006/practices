CSS display preperty.

  1. display: inline

    it makes the block element inline element.It only takes the space required for content, leaving the rest space for other elements to come. Setting other dimension properties isnt like width, height, margin or padding is not allowed in Inline Display.

    for example:
    I am a div and I am a block element.

  2. dislay: block

    It takes the full width available across the website page leaving a new line before and after the element.
    EXAMPLE:

    this is the span and it is the inline element.

  3. display: inline block

    This property is quite similar to inline but dimension properties are allowed in the case of the inline-block display. refer the tutorial @4:30

    EXAMPLE:

    this is a para block element

    this is the para 2 block element

    note: if in the style tag there is no inline-block then the margin and the padding will not be aplicable from the top.
    also the width will not be applicable in the first conditon.


  4. display: none

    when this property is applied the browser will remove the visibility and the space of that container or element.

    display none EXAMPLE:

    It hides the visibility of the container or the element without removing the space occupied by that element.
