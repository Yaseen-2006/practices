CSS float and clear.

  1. float.
    the float is used in the styles of the image tags to allow the content to be follwed by the given direction.

    for example: here is the container named cards containing the 4 more divs named the respective numbers of the cards.

    syntax: 

    img{
        float: left;
                        }

    img{
        float: right;
                        }

    but here the picture overflows from the parent border if the content inside the border is not much. as the solution we use display: flow-root; in the parent div. in the parent div's style. syntax: 
    img{
        display: flow-root 
                                }
  2. clear.
    in the clear the the direction like float is also given here but to the element. that element can be a div or a paragraph.
    the to which the clear is applied it says :
    "i don't want any of the things on my right side"
    example: 
    img{
    float: right;
                    }

    but here the picture overflows from the parent border if the content inside the border is not much. as the solution we use display: flow-root; in the parent div. in the parent div's style.
    it has several values such as clear: right, left, both.
