CSS positions

  1. static positions: syntax:: position: static;
    this position is the default position of any div we include in the html page. this does not allow us to change the position of the div.
    example::
    this div has the static position..
    in this situation we can't change the height and the z-index of the given div.. and can't move the div left or right.
  2. relative position: syntax:: position: relative;
    in this position of the div we can change the z-index and the top bottom and right and left positions.
    example::
    this div has static position
    this div has relative position.
    syntax of above things:: static div:: position: static;
                                           relative div:: position: realative; 
    background-color: aqua; 
     position: relative; 
     z-index: 1; 
     top: -50px; 
     left: 50px;
  3. absolute position:: syntax:: position: absolute;
    in this position the box in which the position property is applied that box will try to adjust its position according to the nearest positioned ancestor i.e the nearest positioned tag.
    example::
    this is the parent div and have the position relative.
    this div has the static position.
    this div has the absolute position. so the position applied to it will be adjusted according to the nearest positioned container i.e the parent box
    the box behind has the static position (by default)
    if i would not give the position relative to the parent box something like this could happen::

    this div has the absolute position. so the position applied to it will be adjusted according to the nearest positioned container i.e the parent box have the position static so it would adjust the position according to its parent's parent div because i have set it on relative position
    Exception: : transform, filter, or perspective properties can also make an element appear as positioned.
    transform: translate(0);
    filter: invert(0);
    perspective: 0em;

  4. Fixed position: syntax:: position: fixed;
    in the fixed position when the div is given the position fixed it dos not move even we scroll..
    example::
    this is the fixed div. having the position:fixed;
