CSS variables.

    CSS variables are generally used to change the color of the website where we want to change the color of the website at multiple places simultaneously. there are two types of variables.
but there are more things we can use as the variables such as padding, margin, opacity,

  1. Global variables.
    the global variables are used to change the color of the website on multiple places and it is declared in the style tag ,
    declaration syntax: 
    :root{
    --color: blue;
    --seccolor: lightblue;

    and then called in the address where it will be applied:
    example::
    I am the div that has colored using the global variables. 
                            
                            :root {
                                --color: blue;
                                --seccolor: lightblue;
                                --txtcolor: white;
                            }
                            .div3{
                                width: 40vw;
                                height: 10vw;
                                border: 2px solid red;
                                text-align: center;
                                padding: 12px;
                                box-sizing: border-box;
                                background-color: var(--color);
                                color: var(--txtcolor);
                                border-color: var(--seccolor);
                            }

    we can use these variables anywhere in the website to apply these colors in the backgroun, text, and averywhere the color is applied. We make the global variable in the :root{ } which will be declared.
  2. local variables.
    local variables are the varibles that are declared in the individual destinations where the global variables are declared. we can also overwrite the variables and give them different colors and that color will be applied to the desired destination.
    example::
    i am the div colored using the local variables. 
                            :root {
                            --color: blue;
                            --seccolor: lightblue;
                            --txtcolor: white;
                        }
                            .div4{
                            width: 40vw;
                            height: 10vw;
                            border: 2px solid red;
                            text-align: center;
                            padding: 12px;
                            box-sizing: border-box;
                            background-color: var(--color);
                            color: var(--txtcolor);
                            border-color: var(--seccolor);
                            --color: rgb(233, 140, 140);
                            --seccolor: rgb(100, 3, 3);
                            --txtcolor: rgb(49, 16, 16);
                        }
    3. exception:: when the variable is not found then the color will be applied which is given next to it
    example:: 
    .div{
            backgroun-color: var(--color, blue);
                }
    here when the variable "--color" will not be foun then the blue color will be applied.
