CSS trasitions

  1. transition-property.
    this property is used to define the specific properperty on which we have to apply the transition.
    It has the several values such as::
    NONE, ALL, and some specific property such as (width, height, scale, transform etc.)
    syntax:: 
    • transition-property: none;
    • transition-property: all;
    • transition-property: transform, background-color;
    • transition-property: width, height;
  2. transition-duration.
    using this transition property we can decide the duration of the transition. (in how much time the transition will be completed.)
    syntax:: 
    • transition-duration: 1s;
    • transition-duration: 2s;
    • transition-duration: 3s;
    • and so on.........
  3. transition-timing-function.
    using this property we can decide the the behaviour of the transition. It has many values.
    some of them are:: ease-in, ease-out, ease-in-out, ease, cubic-bazier (this is the custom behaviour of the trasition we can customise the transition behaviour according to our need.)
    cubic-bazier: this property can be customised in the browser. syntax:: 
    • transition-timing-function: ease-in;
      •  
    • transition-timing-function: ease-out;
      •  
    • transition-timing-function: ease-in-out;
      •  
    • transition-timing-function: ease;
      •  
    • transition-timing-function: cubic-bezier(0.075, 0.82, 0.165, 1);
      •
  4. transition-delay.
    this property specifies that the delay in the biginning of the trastion i.e after how much time the transition will begin.
    in this property the value are entered in seconds i.e. 1s, 2s, 3s
    syntax:: 
    • transition-delay: 1s;
      •  
    • transition-delay: 2s;
      •  
    • transition-delay: 3s;
      •
    • and so on.......
  5. transition shorthand (transition: [all transition properties.];)
    using this property we can include all the properties in one line.
    in this property those all the four properties fit in one property and they have their specified places in that bracket.
    example-syntax::
    transition: (transition-property) (transition-duration )(transition-timing-function) (transition-delay);
    one more example-syntax::
    transition: all 2s ease-in-out 1s;
